# NutriDePP 🍎🏋️‍♂️ — Alimentación y Deporte

NutriDePP es una plataforma web interactiva diseñada para conectar el bienestar nutricional con el rendimiento deportivo. El sistema permite gestionar de manera integral la alimentación y las actividades diarias, ofreciendo una experiencia dinámica para usuarios enfocados en optimizar su salud y rendimiento físico.

---

## 🚀 Características Principales

Basado en la estructura del proyecto, la plataforma ofrece las siguientes funcionalidades:

- **Gestión Dinámica de Recetas (`recetas.html` / `recetas.json`):** Sistema interactivo que procesa y muestra un catálogo estructurado de recetas enfocadas en alimentación deportiva y saludable.
- **Planificación de Actividades (`actividades.html` / `calendario.html`):** Módulos dedicados a la organización temporal de rutinas físicas, entrenamientos y comidas.
- **Acceso y Seguridad de Usuarios (`login.html` / `registro.html` / `contraseña.html`):** Flujo completo de autenticación y registro para brindar una experiencia personalizada y segura.
- **Página Principal e Interacción (`index.html` / `contacto.html`):** Panel de bienvenida intuitivo que unifica el ecosistema y un canal de comunicación directo.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructuración semántica de cada sección, asegurando accesibilidad en formularios y contenidos.
- **CSS3 (`estilos.css`):** Diseño visual unificado, responsivo y moderno, optimizado para la experiencia del usuario (UX) en interfaces de bienestar.
- **JavaScript (`script.js`):** Lógica del lado del cliente para manejar la interactividad, la manipulación del DOM y la carga dinámica de datos.
- **JSON (`recetas.json`):** Almacenamiento estructurado de la base de datos de recetas alimenticias para su consumo ágil en la aplicación.
